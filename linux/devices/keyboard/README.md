
 ## ساخت کلید جایگزین برای ↓


می‌تونی مثلاً کلید F10 یا CapsLock یا هر چیزی رو تبدیل به Arrow Down کنی.

 ### با xmodmap (ساده و سریع):

اول نام کلید مورد نظر رو پیدا کن:

```bash
xev | grep keycode
```

یه پنجره باز می‌شه؛ کلید مورد نظر (مثلاً F10) رو بزن و عدد keycodeش رو یادداشت کن.
فرض کنیم keycode 76 بود.

حالا اون کلید رو به ↓ نگاشت کن:
```bash
xmodmap -e "keycode 76 = Down"
```

تست کن؛ اگه درست شد، برای دائمی کردنش، این دستور رو بنداز توی فایل:
```bash
~/.Xmodmap
```

و داخلش همین خط رو بنویس.
